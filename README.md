# 🌿 Shraddha's Mehndi Portfolio

A beautiful portfolio website to showcase original Mehndi / Henna art designs.

## Run Locally
```
npm install
npm start
```
Then open: http://localhost:3000

## Deploy Free on Railway (Recommended)
1. Push this project to GitHub
2. Go to https://railway.app → Login with GitHub
3. Click "New Project" → "Deploy from GitHub repo"
4. Select this repo → Railway auto-detects Node.js
5. Your site goes live with a public URL!

## Deploy Free on Render
1. Push to GitHub
2. Go to https://render.com → New → Web Service
3. Connect your GitHub repo
4. Build Command: `npm install`
5. Start Command: `npm start`
6. Click Deploy → get a free public URL!
